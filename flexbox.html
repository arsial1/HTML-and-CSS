When flexbox was developed a couple of years ago, the main idea behind it was to basically allow browsers to automatically divide empty space inside some container element by its child elements. And by doing that, the browser takes away a lot of work that previously the web developers had to do.

In this post, let's focus on Flexbox. We will learn what Flexbox is all about, and why you too should learn and use it.

What is Flexbox?

Flexbox, short for Flexible Box Layout, is a one-dimensional layout model in CSS that offers a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic.

It's a powerful tool designed to accommodate complex layouts and alignments that were difficult to achieve with traditional CSS properties alone. Flexbox makes it easier to design flexible responsive layout structures without having to use floats or positioning.

You can explore the Flexbox guide at CSS-Tricks to learn more.

How to use Flexbox?

There are not many concepts behind flexbox that you need to understand in order to start using it. But here is what you do need to know.

So, first of all, the element on which we want to use flexbox is called the flex container. All we have to do in order to create a flex container is to set display property of container to flex. And that's it. 

          
            
            .container {
                display: flex;
            }

if we do this, then all the direct children of that flex container will become the so-called flex items.

By default flex items are aligned to main axis (horizontal) but we can change the direction of flexbox axis from horizontal to vertical, so we can align elements/flex items along the main axis and along the cross axis in different ways.

We always need to know which axis we are dealing with.

Overview of all Flexbox properties

Overview of all the CSS properties that are part of the flexbox specification both for the flex container and for the flex items.

So for the flex container, the most important ones are the gap, the justify-content and the align-items properties. 

          
            
            .container {
                display: flex;
                align-items: center;
                justify-content: space-between;
                gap: 20px;
            }

There are three other properties that we can apply on the flex container element, which are flex-direction, flex-wrap, and align-content.

In properties for flex-items, align-auto is pretty important because it can override the align-itemsproperty

Then we have these other ones which are flex-grow, flex-shrink, flex-basis. flex is the recommended shorthand for flex-grow, flex-shrink and flex-basis.

Flexbox allows flex items to flex (grow or shrink) to fill available space in the container. This is particularly useful for creating responsive designs that adjust and adapt to different screen sizes.

Why should you learn Flexbox?

There are countless and compelling reasons for learning and using Flexbox:

  • Responsive Design: Flexbox makes it much easier to create responsive designs that adapt to different screen sizes and devices. This is essential in today's world where users access websites on a variety of devices, from smartphones to desktop computers.
  • Simplified Layouts: Flexbox simplifies complex layout tasks that were traditionally challenging with CSS alone. With Flexbox, you can easily create complex layouts with less code, making your development process more efficient and your code easier to maintain.
  • Flexibility and Control: Flexbox gives you fine-grained control over the alignment, ordering, and sizing of elements within a container. This level of control allows you to create visually appealing designs that were previously difficult to achieve.
  • Future-Proofing: While newer layout methods like CSS Grid are gaining popularity, Flexbox is still widely used and will continue to be relevant for the foreseeable future. By mastering Flexbox, you'll have a solid foundation in modern web layout techniques that will serve you well in your career.

Flexbox Summary

  • Flexbox makes it easy to automatically align items to one another inside a parent container, both horizontally and vertically
  • Flexbox solves common layout problems such as vertical centering and creating equal-height columns
  • Flexbox is perfect for replacing floats, allowing us to write fewer and cleaner HTML and CSS code
